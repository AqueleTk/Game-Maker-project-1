# Jogo do RPG Internato Merlin — Presente Bell

Nesse jogo, a protagonista é Lillian, uma caçadora de fantasma que precisa capturar alguns fantasmas para livrar a escola dos espíritos malignos, mas está sozinha.

Objetivo: Jogo estilo Rouguelike com investigação, com sistemas de capturas, upgrades, nível, armas diferentes e alguns rituais e boss fight.
Extra: Sistema de pacto, mais personagens jogaveis, cenário bonito, cutsenes, menu verdadeiro, dialogos.

O jogo tem apenas o básico do básico, ainda no processo de criação, tendo apenas:
* Movimentação
* Pixel Perfect
* Lillian Place Holder
* Colisão com parede
